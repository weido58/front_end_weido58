## 一、CSS简介

```
美化网页，布局页面
```

```
CSS:层叠样式表
```

```
HTML：格式
CSS:样式
```

1.语法

```
由两部分构成：
选择器以及一条或多条声明
```

```
选择器是用于指定CSS样式的HTML标签，花括号内是对该对象设置的具体样式。
多个键值对之间用分号区别。
```

2.代码风格

```
展开样式
小写字母
属性值前，冒号后面保留一个空格
选择器和大括号中间保留空格
```

## 二、CSS基础选择器

1.作用

```
选择标签
```

2.选择器分类

```
1.基础选择器
由单个选择器组成，包括：
标签选择器、类选择器、id选择器和通配符选择器
2.复合选择器
```

2.1标签选择器

```
是指用 HTML标签名称 作为选择器。
```

```
把某一类标签全部选出来
```

2.2 类选择器（最常用）

1.简介

```
单独选一个或者某几个标签
```

```
语法：
.类名 {
	属性: 属性值1；
	...
}

注意：
需要class属性来指定类
```

```
class：
长名称或词组用中横线命名
尽量全使用英文
```

2.多类名

```
可以给一个标签指定多个类名
```

```
使用方式：
1.在class属性中写多个类名
2.多个类中间必须用空格分开
```

```
使用场景：
让类各司其职，统一方便修改
```

2.3 id选择器

```
HTML元素一id属性来设置id选择器，以#来定义
```

```
语法：
#id名 {
	属性1: 属性值1;
	...
}
```

```
和类选择器的区别：
id选择器只能调用一次，经常和js搭配使用
```

2.4 通配符选择器

```
使用*定义，表示选取页面中所有元素
```

```
不需要调用
特殊情况才使用
```

## 三、CSS字体属性

```
fonts（字体）属性用于定义字体序列、大小、粗细和文字样式。
```

1.字体系列

```
font-family
定义文本的字体系列
```

```
各种字体必须使用逗号隔开
如果字体由多个单词组成，加引号
```

2.字体大小

```
font-size: 0px;
```

```
px(像素)
```

```
标题标签<h>比较特殊，需要单独指定大小
```

3.字体粗细

```
font-weight: 
```

```
参数：
normal : 正常的字体。相当于number为400。声明此值将取消之前任何设置 
bold : 粗体。相当于number为700。也相当于b对象的作用 
bolder : IE5+　特粗体 
lighter : IE5+　细体 
number : IE5+　100 | 200 | 300 | 400 | 500 | 600 | 700 | 800 | 900 
```

```
提倡使用数字表示
400等同normal
700等同bold
```

4.文字样式

```
font-style:
```

```
参数：
normal : 正常的字体 
italic : 斜体。对于没有斜体变量的特殊字体，将应用oblique 
oblique : 倾斜的字体 
```

5.字体复合属性

```
body {
    font: font-style font-weight font-size font-family;
}
```

```
不能更换顺序，属性间以空格隔开
必须保留font-size和font-family,其它可省略
```

## 四、CSS文本属性

```
定义文本的外观，比如颜色、对其、装饰、缩进等等。
```

1.文本颜色

```
color
```

```
#RRGGBB ：CSS1 IE4+ , NS4+ 三个两位十六进制正整数。取值范围为：00 - FF 
rgb ( R,G,B ) ：CSS1 IE4+ , NS4+ 表示红，绿，蓝的正整数或百分数数值 
Color Name ：CSS1 IE4+ , NS4+ 颜色名称。不同的浏览器会有不同的预定义颜色名称。 
```

```
十六进制最常用
```

2.对齐文本

```
text-align
```

```
参数：
center left（默认） right
```

3.装饰文本

```
text-decoration
```

```
none 默认
underline 下划线
overline 上划线
line-through 删除线
```

4.文本缩进

```
text-indent
```

```
将段落的首行缩进
```

```
参数：
0px
0em
```

```
em是一个相对单位，就是当前元素1个文字的大小，若当前文字没有设置大小，则会按照父元素的1个文字大小。
```

5.行间距

```
line-height
```

```
控制文字行与行之间的距离。
行间距=上间距+文本高度+下间距
```

```
参数：
0px
```

## 五、CSS引入方式

1.行内样式表（行内式引入）

```
在元素的标签内的style属性中设定CSS样式
```

```
例：
<div style="color: red">测试</div>
```

2.内部样式表（嵌入式）

```
写到html页面内部
写到style标签中，并写到head标签中
```

3.外部样式表

```
样式单独写到CSS文件中，之后把CSS文件引入html页面中
```

```
1.新建一个.css的样式文件
2.在HTML页面中，使用link标签引入这个文件（写在head中）
<link rel="stylesheet" href="css文件路径">
```
